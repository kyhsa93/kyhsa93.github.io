# Git page jekyll blog

## cli
```bash
  jekyll serve    # start server
  jekyll serve -l # start live server
```

## Site configuration
* sitemap: https://kyhsa93.github.io/sitemap.xml
* defaul config: [/_config.yml](https://github.com/kyhsa93/kyhsa93.github.io/blob/master/_config.yml)
* Gem module version lock file: [/Gemfile.lock](https://github.com/kyhsa93/kyhsa93.github.io/blob/master/Gemfile.lock)
* All post files: [/_posts/](https://github.com/kyhsa93/kyhsa93.github.io/tree/master/_posts)

## Add post
* Add posting date front of file name
* Add layout, title, date, author, category, cover top of file
* If want add image, please add in assets directory
